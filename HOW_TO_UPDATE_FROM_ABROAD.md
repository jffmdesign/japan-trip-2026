# ✈️ 出國時如何更新網頁？ (手機版攻略)

雖然我 (Antigravity AI) 住在您的電腦裡，當您出國沒帶這台電腦時，您還是可以用 **手機** + **任何 AI App** (ChatGPT/Gemini) 來更新網站！

這需要三個步驟：**叫 AI 寫扣 -> 複製 -> 貼上 GitHub**。

## 準備工作 (出國前)
1.  **手機下載 GitHub App** (或者直接用瀏覽器開 GitHub)。
2.  **登入 GitHub** 帳號。
3.  **把這個儲存庫 (Repo) 加入書籤**。
    *   您的網址： `https://github.com/jffmdesign/japan-trip-2026`

---

## 步驟一：叫 AI 寫新的行程 (使用手機 AI)
打開手機上的 **ChatGPT** 或 **Google Gemini** App，輸入指令：

> 「請幫我改一下原本的 HTML 碼，把 Day 3 的晚餐改成『燒肉弘』，直接給我那一段 `<div id="day3">...</div>` 的完整 HTML 就好。」

AI 會生成一段新的 HTML 程式碼，請點擊 **「複製 (Copy)」**。

---

## 步驟二：到 GitHub 貼上程式碼
1.  用手機瀏覽器打開您的儲存庫：`https://github.com/jffmdesign/japan-trip-2026`
2.  點擊列表中的 **`index.html`** 檔案。
3.  點擊右上角的 **... (選單)** -> **Edit file** (或是鉛筆圖示 ✏️)。
    *   *注意：如果手機 App 不能編輯，請用 Chrome 瀏覽器並切換成「電腦版網站」模式。*
4.  找到要修改的地方 (例如 Day 3 的位置)。
5.  **貼上** 剛剛 AI 給您的新程式碼，覆蓋掉舊的。

---

## 步驟三：存檔 (Commit)
1.  滑到頁面最下方 (或右上角) 找到 **"Commit changes"**。
2.  Message 隨便填 (例如：`Update dinner`)。
3.  按下 **Commit changes** 按鈕。

🎉 **完成！** 等 1 分鐘，您的旅遊網站就會更新了。

---

### Tips
*   **出發前測試**：建議您現在先用手機試試看能不能登入 GitHub 並找到 `Edit` 按鈕。
*   **備用方案**：如果您有帶筆電，直接用網頁版 GitHub 編輯會更方便 (有搜尋功能 `Ctrl+F`)。
